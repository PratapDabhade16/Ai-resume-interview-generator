# AI Resume Interview Question Generator

This project generates personalized interview questions based on a user's resume using a Large Language Model (LLM).

## Features
- Resume upload (PDF)
- Role-based interview questions
- Experience & difficulty based customization
- Uses pre-trained LLM (ChatGPT API)

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- AI: LLM (ChatGPT API)

## Project Type
Mini Project (Academic)
